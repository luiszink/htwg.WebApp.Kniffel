About the Kniffel Project

Welcome to the Kniffel Project

This project was developed as part of my studies in Applied Computer Science, with a focus on enhancing skills in web development, software engineering, and user interface design. Kniffel, also known as Yahtzee, is a game of luck and strategy where players aim to score the highest number of points by rolling five dice and matching specific combinations.

Project Overview

This Kniffel game is built using Scala with the Play Framework, combining backend functionality with a clean and interactive frontend experience. One of the primary goals of this project was to integrate a terminal-based User Interface (TUI) into a web application, allowing users to play the game directly through their browser.

Key Features

  • Classic Kniffel gameplay with familiar rules and score combinations.
  • Interactive Web UI: Powered by Scala's Play Framework to ensure smooth and responsive gameplay.
  • TUI Integration: The game offers a command-line style interface within the browser for users who enjoy a retro gaming experience.
  • Scoring Automation: Automatic calculation of scores based on the combinations rolled, following the traditional rules of Kniffel.

Technology Stack

  • Programming Language: Scala
  • Framework: Play Framework
  • Frontend: HTML, CSS, JavaScript (for UI interactions)
  • Backend: RESTful APIs built using Play for handling game logic
  • Development Tools: SBT (Scala Build Tool), Metals for Scala, and Git for version control.

Learning Goals

The Kniffel project is not just a fun game, but a learning experience in applying various software development principles:

  • Software Architecture: Design and development of a modular system where the backend game logic is decoupled from the frontend.
  • Unit Testing: Extensive use of JUnit for testing game logic and ensuring robust code quality.
  • Continuous Integration: Using Git and automated testing to ensure smooth project evolution and maintain code reliability.

Challenges and Future Plans

One of the challenges in this project was integrating the TUI within a modern web interface. Moving forward, I aim to:

  • Improve the UI design and make it more visually appealing.
  • Expand the game with multiplayer support, allowing users to compete against each other in real-time.
  • Implement advanced features like saving and loading games, and leaderboards to track high scores.
